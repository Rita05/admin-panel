# Админ-панель

## Стек используемых технологий
- **React 18+** 
- **TypeScript / JavaScript** 
- **MUI** 
- **Redux Toolkit**  
- **Axios** 

## Запуск проекта

###  1.Склонировать проект 
git clone https://github.com/Rita05/admin-panel.git

###  2.Установить зависимости

### `yarn install`

###  3.Запустить проект 

### `yarn start`
